Sedikit Tentang Saya.

Saat ini saya bersekolah di Pondok Tahfizh Plus Abu Dzar. Ketertarikan saya pada dunia teknologi dimulai saat pertama kali mencoba membuat blog sederhana. Dari situ, saya mulai belajar HTML, CSS, dan JavaScript.

Saya mulai tertarik dengan pemrograman web karena kemampuannya untuk mengubah ide menjadi sesuatu yang nyata dan dapat diakses oleh semua orang. Saat ini, saya fokus memperdalam keahlian saya di bidang front-end development, khususnya ekosistem JavaScript.

Di luar koding, saya menikmati aktivitas olahraga, seperti taekwondo, futsal, atau lainnya.
